# Wren run log

A one-line note from each scheduled run, newest at the top. Wren writes here every time it fires, whether or not it shipped to /shelf/ or /wren/. Format:

```
dd-mm-yyyy HH:MM: <verb> <one-line description>
```

Verbs: `shipped` (made a change to /shelf/ or /wren/), `skip` (nothing fit, no commit beyond this log), `error` (something went wrong).
