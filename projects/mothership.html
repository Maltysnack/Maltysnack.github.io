Mothership

Built in LÖVE2D (Lua) · 2D strategic card game

Status In development
Engine LÖVE2D
Platform Desktop

Mothership is a digital adaptation of a physical strategic card game, rebuilt from the ground up in LÖVE2D. The core loop is roguelike — each run you build a deck, fight through encounters, and make choices that compound. The battle system adapts the original physical mechanics faithfully while adding new digital-only features: animated card states, visual effects for abilities, and a persistent run structure. The card renderer has been the main beast to tame, but the bones are solid.

Devlog #1

Core loop is alive

10 Feb 2025

Got the core battle loop working and the roguelike structure drafted. Cards can be played, effects resolve, and turns pass correctly. The card renderer is being fussy with layering — masked sprites with dynamic colour fills are trickier in LÖVE than I expected — but progress is progress. Next milestone: the full first encounter chain, then a basic meta-progression layer.
