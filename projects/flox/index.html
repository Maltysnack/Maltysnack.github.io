Project

Flox

A small language for describing flow networks. You write nodes and arrows; it simulates how stuff moves through them over time.

v0.1Java06-05-2026

Example

system Canberra;

Googong[2, 2.0, 0.7] -> googongDam{1000, 80%, 0.8, 1.2} -> Queanbeyan[3, 1.5];
upperMongolo[3, 1.8, 0.6] -> Queanbeyan[3, 1.5];
Queanbeyan[3, 1.5] -> centralMongolo[];
centralMongolo[] -> burleyGriffinDam{5000, 60%, 0.9, 1.5} -> lowerMongolo[];

map Canberra;
print Canberra 10;

Square brackets are rivers. Curly brackets are dams. Empty brackets are outlets. -> means flow from left to right. print Canberra 10 simulates ten days and prints the flow table.
