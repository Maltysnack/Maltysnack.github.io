Project

Flox

A small language for describing flow networks. You write nodes and arrows; it simulates how stuff moves through them over time.

v0.1Java06-05-2026

Example

Program

system Demo;

Spring[2, 2.0, 0.7]    -> Confluence[3, 1.8];
Tributary[1, 1.5, 0.6] -> Confluence[3, 1.8];
Confluence[3, 1.8]     -> Outlet[];

map Demo;
print Demo 5;
