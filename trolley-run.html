Trolley Run

Engine TBD · Arcade racing · WASD / Controller / Mobile tilt

Status Concept phase
Genre Arcade racing
Platform Desktop / Mobile (TBD)

You are not in a car. You are pushing a trolley. All four wheels rotate independently, the physics are deliberately chaotic, and staying on course is the entire challenge. Trolley Run is a top-down arcade racer where the vehicle itself is the obstacle — each turn fights you, each bump compounds, and loading up the trolley unevenly shifts the centre of gravity in ways that will make you laugh and then immediately swear. Different environments swap between shopping trolleys, warehouse pallet movers, and whatever else feels appropriately absurd. Playable with WASD, a controller, or phone tilt — though tilting is especially unhinged.

🛒
Four-wheel physics

Each wheel steers independently. Small inputs have cascading effects. Control is earned, not given.

📦
Load it up

Items in the trolley shift your weight distribution. Stack wrong and you're fighting physics the whole run.

🏬
Environments

Supermarket aisles, warehouse floors, car parks. Each level changes what kind of trolley you're running.

📱
Tilt controls

On mobile you physically tilt the device to steer. It makes an already chaotic game legitimately unplayable in the best way.
